# 05. UI仕様

## 画面名
LINE風トーク吹き出し画像ジェネレーター

## 画面構成
```text
+--------------------------------------+
| LINE風トーク吹き出し画像ジェネレーター |
+--------------------------------------+

[テキスト入力欄]

吹き出しタイプ:
(●) 自分（緑）
( ) 相手（白）

[プレビュー]

[PNGダウンロード]
[JPEGダウンロード]
```

## UI要素
### 1. タイトル
`LINE風トーク吹き出し画像ジェネレーター`

### 2. 説明文
`文章を入力すると、LINE風の吹き出し画像を生成できます。PNGは透明背景、JPEGは白背景で保存されます。`

### 3. テキスト入力
- Streamlit: `st.text_area()`
- 高さ: 150px程度
- 最大文字数: 1000文字
- プレースホルダー: `ここにメッセージを入力してください`

### 4. 吹き出しタイプ選択
- Streamlit: `st.radio()`
- 選択肢: 自分（緑） / 相手（白）
- 初期値: 自分（緑）

### 5. プレビュー
入力がある場合、画像をプレビュー表示する。MVPでは入力変更に応じて即時プレビューでもよい。

### 6. ダウンロードボタン
- PNGダウンロードを優先表示
- JPEGダウンロードも表示
- ファイル名例: `line_bubble.png`, `line_bubble.jpg`

## バリデーション
### 空文字
`メッセージを入力してください。`

### 1000文字超過
`1000文字以内で入力してください。`

## UI方針
- シンプルにする
- 設定項目を増やしすぎない
- MVPではデザイン調整項目を出さない
