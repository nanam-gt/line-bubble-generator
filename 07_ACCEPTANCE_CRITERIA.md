# 07. 受け入れ条件

## 起動
- [ ] `streamlit run app.py` でアプリが起動する
- [ ] ブラウザで画面が表示される
- [ ] エラーなく初期画面が表示される

## 入力
- [ ] 日本語を入力できる
- [ ] 改行入りテキストを入力できる
- [ ] 空文字の場合は警告が出る
- [ ] 1000文字超過時に警告が出る、または生成を抑止する

## 吹き出しタイプ
- [ ] 自分（緑）を選択できる
- [ ] 相手（白）を選択できる
- [ ] 自分を選ぶと緑吹き出しになる
- [ ] 相手を選ぶと白吹き出しになる

## 画像生成
- [ ] 1つの吹き出し画像が生成される
- [ ] 背景なしで生成される
- [ ] PNGは透明背景になる
- [ ] JPEGは白背景になる
- [ ] 吹き出しが画像端で切れない
- [ ] 外側余白が最小限である
- [ ] しっぽが描画される
- [ ] 自分側は右向きしっぽになる
- [ ] 相手側は左向きしっぽになる
- [ ] 相手側の白吹き出しが透明背景でも視認できる

## テキスト
- [ ] 日本語が文字化けしない
- [ ] テキストが吹き出し内に収まる
- [ ] 長文が自動改行される
- [ ] ユーザー入力の改行が維持される
- [ ] テキスト色が黒である

## ダウンロード
- [ ] PNGをダウンロードできる
- [ ] JPEGをダウンロードできる
- [ ] ダウンロードしたPNGが透明背景である
- [ ] ダウンロードしたJPEGが白背景である
- [ ] ファイルが画像ビューアで開ける

## MVP完了判定
- [ ] テキスト入力
- [ ] 自分/相手選択
- [ ] LINE風吹き出し生成
- [ ] 透明PNG保存
- [ ] 白背景JPEG保存
- [ ] 日本語表示
- [ ] 自動改行
- [ ] しっぽ描画
