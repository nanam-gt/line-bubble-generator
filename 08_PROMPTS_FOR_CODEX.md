# 08. Codex用プロンプト例

## 初回実装プロンプト
以下をVS CodeのCodexに貼り付けてください。

```text
Python + Streamlit + Pillowで、LINE風トーク吹き出し画像ジェネレーターを実装してください。

要件:
- Webブラウザ上で動くStreamlitアプリにする
- 入力した文章をLINE風の吹き出し画像にする
- 1画像につき1吹き出しのみ
- 「自分（緑）」と「相手（白）」をラジオボタンで選択できる
- 自分側は緑の吹き出し、右向きしっぽ
- 相手側は白の吹き出し、左向きしっぽ
- 背景は不要
- PNGは透明背景
- JPEGは白背景
- PNG/JPEGをダウンロードできる
- 日本語対応
- Noto Sans JPを使う想定。ただしフォントが見つからない場合はフォールバックする
- 長文は自動改行する
- ユーザー入力の改行も維持する
- 画像サイズは固定ではなく、吹き出しに合わせた最小サイズにする
- 外側余白は5〜10px程度
- 相手側の白吹き出しには薄い枠線を付ける

作成ファイル:
- app.py
- image_generator.py
- requirements.txt
- README.md
- .gitignore

コードは読みやすく、関数を分けてください。
画像生成ロジックは image_generator.py に分離してください。
StreamlitのUIは app.py に書いてください。
```

## 画像生成だけ改善したい場合
```text
image_generator.pyの吹き出し描画を改善してください。

改善したい点:
- 角丸を自然にする
- しっぽの位置を自然にする
- 日本語テキストの中央揃えを改善する
- 白吹き出しの枠線を自然にする
- PNG透明背景を維持する
- JPEG変換処理を壊さない

既存の関数構成はなるべく維持してください。
```

## 自動改行を改善したい場合
```text
現在の文字数ベース改行を、Pillowのテキスト描画幅を使った改行処理に改善してください。

要件:
- 日本語対応
- 英数字混在対応
- ユーザー入力の改行を維持
- 最大幅を超えないように折り返し
- 既存UIは変更しない
```

## UIを整えたい場合
```text
Streamlit UIを使いやすく整えてください。

要件:
- タイトルと説明文を分かりやすくする
- テキスト入力欄を見やすくする
- PNGダウンロードをメインに見せる
- JPEGダウンロードも残す
- 未入力時の警告を分かりやすくする
- 仕様は変えない
```
